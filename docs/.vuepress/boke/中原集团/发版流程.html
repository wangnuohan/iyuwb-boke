    • 发版方法：

      • 打包项目npm run build

      • 把生成的.nuxt文件夹打包压缩成uatXXXX.zip,XXXX 为当天日期

      • 打开 Xshell，进入 UAT-WAP 会话(界面如下)

        • 进入目标文件夹730-m-web: cd 730-m-web

        • 上传压缩文件：sudo rz

        • 解压文件：sudo unzip uatXXXX.zip

        • 解压提示：A （覆盖所有文件）

        • 重新启动 pm2 进程： pm2 restart all
